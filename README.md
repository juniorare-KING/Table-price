Ce projet est une page de tarification (Pricing Table) responsive réalisée en HTML5 et CSS3.

Intégration fidèle d'une maquette graphique.

Utilisation de Flexbox/Grid pour l'alignement des offres et des fonctionnalités.

Gestion des icônes SVG.

Site adaptatif (Responsive) pour mobile et desktop.

Code versionné avec Git et déployé via GitHub Pages."
